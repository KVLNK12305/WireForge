# WireForge

> Exploring networking, protocol internals, and network security through analysis, implementation, and experimentation.

## About

WireForge is my networking lab.

This repository documents my journey into computer networks—from understanding protocol fundamentals to building networking and security tools in Rust. The focus is on learning by doing: studying protocols, analyzing real packet captures, and implementing concepts from scratch.

Rather than treating networking as theory alone, every topic is explored through:

- 📖 Protocol study
- 📡 Packet analysis with Wireshark
- 🦀 Rust implementations
- 🔒 Security implications
- 🧪 Hands-on experiments

## Philosophy

I believe the best way to understand a protocol is to:

1. Learn why it exists.
2. Observe it in real network traffic.
3. Implement part of it.
4. Understand how it can be attacked, defended, or optimized.

This repository is a long-term engineering notebook that grows alongside my knowledge.
